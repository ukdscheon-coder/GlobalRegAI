## 1. 프로젝트 기획서 (운영자 보관용)

### 1.1 프로젝트 목표
* **비전**: 전 세계 규제 기관(FDA, EMA, 식약처 등)의 복잡한 인허가 과정을 AI로 완전 자동화하는 B2B SaaS 플랫폼.
* **핵심 가치**: 규제 전문가(RA)의 업무 시간을 90% 단축하고, 전문적인 다국어 법적 용어(RAG 엔진)를 통해 정확도 99% 달성.

### 1.2 배포 및 100% 무료 인프라 (Zero-cost Architecture)
초기 자본 지출(비용)을 0원으로 유지하면서 수익을 창출하는 아키텍처입니다:
* **웹 호스팅**: GitHub Pages (완전 무료 배포)
* **도메인**: 보유 중인 `globalregai.info` 활용
* **데이터베이스 및 인증**: Supabase (Free Tier 활용, 500MB DB 및 5만 명 유저 무료)
* **결제 시스템**: Stripe (가입비/월유지비 없음, 결제 발생 시에만 소액 수수료 공제)

## 2. 앱 내 사용자 제공용 공식 설명서 (User Manual)
*(아래 내용은 실제 앱 화면의 "Help" 또는 "가이드" 메뉴에 들어갈 내용입니다)*

### 🚀 Welcome to GlobalRegAI!
의료기기, 의약품, 화장품 글로벌 진출을 위한 최고의 AI 규제 파트너, GlobalRegAI에 오신 것을 환영합니다.

#### 주요 기능 안내
1. **글로벌 규제 챗봇 (Global Chat)**:
   미국 FDA, 유럽 EMA, 한국 식약처(MFDS) 등 17개 이상 주요 기관의 최신 법령을 기반으로 질문에 답해드립니다. 
   *(예시 질문: "소프트웨어 의료기기(SaMD)의 FDA 510(k) 심사 절차를 알려줘")*

2. **자동 문서 작성 (Auto-Drafting)**:
   제품의 스펙이나 사용 목적을 입력하시면, AI가 해당 국가의 공식 문서 양식(예: 기술문서, 임상평가 보고서)에 맞춰 전문 용어로 초안을 작성해 드립니다.

3. **다국어 전문 번역 (Legal Translation)**:
   일반 번역기가 아닙니다. 일본 PMDA, 중국 NMPA의 규정 문서를 각 국가의 '공식 규제 용어 사전'을 바탕으로 가장 정확하게 번역합니다.

#### 구독 및 요금 안내 (Pricing)
* **Community (무료)**: 기본적인 AI 규제 상담 기능을 제공합니다.
* **Professional (월 $49)**: 무제한 규제 문서 번역, 최신 변경 사항 알림 메일, 문서 자동 작성 기능 월 100회.
* **Enterprise (별도 문의)**: 기업 맞춤형 내부 규정 학습, 보안 감사(ISO 13485) 가상 시뮬레이션 시스템 제공.

## 3. 관리자가 지금 해야 할 배포(퍼블리싱) 가이드

이제 완성된 코드를 인터넷에 **무료로 공개**하여 사용자들이 접속하고 돈을 내게 만들 차례입니다.

### Step 1: 최고 관리자 계정 생성
1. 앱 화면(로그인 창)에서 **`uk.dscheon@gmail.com`** 이메일로 **[Sign Up(회원가입)]**을 진행하세요.
2. 코드가 이 이메일을 인식하여, 자동으로 결제창을 띄우지 않고 **최고 관리자(Admin) 모드**로 접속시킵니다.

### Step 2: 깃허브를 통한 완전 무료 웹 배포 (GitHub Pages)
코드는 이미 깃허브에 올라가 있습니다. 단 한 줄의 명령어로 인터넷에 배포할 수 있습니다.
터미널에서 다음을 실행하면 끝납니다 (이후 자세한 안내가 이어집니다):
```bash
npm run deploy
```

### Step 3: 결제 시스템 (Stripe) 연동 준비
나중에 사용자들이 결제할 수 있도록 [Stripe.com](https://stripe.com) 에 가입해 두시면 됩니다. 초기 비용은 완전 무료입니다.
