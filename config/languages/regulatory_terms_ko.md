## 의료기기

### 등급 분류
| 등급 | 위해도 | 예시 |
|------|--------|------|
| 1등급 | 낮음 | 혈압계 줄자, 의료용 거즈 |
| 2등급 | 보통 | 혈당측정기, 의료용 초음파 |
| 3등급 | 높음 | 인공관절, 인공수정체 |
| 4등급 | 매우 높음 | 심장박동기, 이식형 제세동기 |

### 인허가 절차
| 용어 | 설명 |
|------|------|
| 품목허가 | 3·4등급 의료기기 허가 신청 |
| 품목인증 | 2등급 의료기기 인증 신청 |
| 품목신고 | 1등급 의료기기 신고 |
| 기술문서 | Technical File — 제품 안전성·성능 입증 서류 |
| 임상시험계획승인 (IND) | 의료기기 임상시험 사전 승인 |
| 적합인정 | GMP 기반 제조소 품질 인정 |

### 주요 서류
- **기술문서** — 설계개발, 위험관리, 성능시험 결과
- **생물학적 안전성 평가** — ISO 10993 기반
- **전기안전 시험성적서** — IEC 60601-1
- **임상평가보고서** — 임상 데이터 또는 문헌 근거
- **라벨링 초안** — 사용설명서 포함

## 의약품

### 허가 유형
| 용어 | 설명 |
|------|------|
| 신약허가 | 새로운 성분·제형 의약품 허가 |
| 제네릭 허가 | 오리지널과 동등성 입증 후 허가 |

| 희귀의약품 지정 | Orphan drug 지정 혜택 |

### GMP 관련 용어
| 한국어 | 영어 | 설명 |
|--------|------|------|
| 의약품 제조·품질관리기준 | GMP | Good Manufacturing Practice |
| 적합판정 | GMP Compliance | GMP 심사 통과 |
| 밸리데이션 | Validation | 공정·시험법 검증 |
| 일탈 | Deviation | 절차 미준수 사항 |
| 부적합 | Non-conformance | 규격 불충족 |
| 시정조치 예방조치 | CAPA | 재발 방지 조치 |
| 제조기록서 | Batch Manufacturing Record | 배치별 제조 기록 |
| 품질검토보고서 | Product Quality Review (PQR) | 연간 품질 검토 |
| 규격외결과 | Out of Specification (OOS) | 시험 규격 초과 |

## 화장품

### 분류 및 허가
| 구분 | 내용 |
|------|------|
| 기능성화장품 | 미백, 주름개선, 자외선차단 등 — 심사 필요 |
| 일반화장품 | 책임판매업자 신고 후 판매 |
| 천연·유기농화장품 | 인증 기준 별도 존재 |

### 주요 용어
- **책임판매업자** — 화장품 제조·판매 책임 주체
- **전성분 표시** — 모든 원료 INCI명으로 표기 의무
- **안전성 자료** — 독성, 안정성, 효능 데이터
- **화장품 제조업** — 제조 시설 등록 필요
